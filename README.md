# Task Manager App
1. First Set Environment Variable,
  -> Powershell:
    - $env:port=anyNumber
  -> CMD:
    - set port=anyNumber& npm start
  
2. In Google,
  -> type URL: 
    - Ex:- Entered Number is: 5000
    - http://localhost:5000

3. Run
  -> Type Command 
    - npm start
    - node app.js
    - nodemon app.js (after installing nodemon)
      - to install nodemon,
        - npm i nodemon OR npm install nodemon
